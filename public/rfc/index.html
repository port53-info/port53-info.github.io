DNS RFC Repository

DNS turns 40 this year! We maintain an up-to-date repository of DNS-related RFCs. To help community members quickly gather documents of interest, we also give finer-grained labels under particular topics (e.g., DNSSEC, IDNA, DNS privacy) to each RFC.

What’s within the list:

  • Documents subsequent to RFC 882 (November 1983)
  • DNS specification: namespace, name registration, DNS protocol
  • DNS operation: functionalities and recommendations about running DNS services
  • DNS security: security extensions, threat analysis, and proposed solutions
  • DNS use: Internet functionalities that explicitly rely on the DNS (e.g., SPF, DANE)
  • Obsoleted/deprecated versions of the above

The list updates once a month. It now includes 444 RFCs under 32 labels.

Comments or suggestions are extremely welcome! Drop us an email (luchaoyi@tsinghua.edu.cn) if you think an RFC is missing, mis-labeled, should be removed, or if you have any other feedback.
